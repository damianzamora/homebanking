# homebanking 

Mi primer proyecto en JAVA consiste en la realizacion de un Homebanking , el cual permite iniciar sesion o crear una cuenta en el banco, ver y crear cuentas bancarias ,dentro de esas cuentas bancarias se pueden ver los movimientos, se puede realizar transferencias entre cuentas propias o terceras, permite altas y bajas de tarjetas de credito , solicitar pr√©stamos y realizar pagos.

El proyecto se encaro utilizando las siguientes tecnologias: JAVA 11, VUE 3.0 , AXIOS , CSS , HTML5 , BOOTSTRAP.

Link para ver proyecto : https://worldbanking.herokuapp.com/Web/index.html
